# Pokemon Adventure
En este proyecto hemos desarrollado una aventura narrativa interactiva que se ejecutará en el terminal. Las decisiones del usuario afectarán tanto al inventario disponible como al final alcanzado.

## Características principales
* Elección de pokemon
* Elección de inventario
* Diferentes rutas que tendrán distinta dificultad dependiendo del pokemon elegido
* Evolución de Pokemon mediante piedras especiales
* Sistema de combate de pokemons
* Modificación o/y utilización de inventario
* Finales diferentes dependiendo de las decisiones anteriores

## Tecnologías utilizadas
* Java 17 y Java 21
* IntelliJ IDEA
* Git y GitHub

## Ejecución del proyecto
1. Abre el proyecto en tu IDE preferido
2. Compila el proyecto
3. Ejecuta la clase "HistoriaPrincipal"

## Requisitos
* Java 17 o Java 21
* IDE con soporte para proyectos Java
